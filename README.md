Objective

This project demonstrates foundational to intermediate SQL skills using the "Sample - Superstore" dataset. The goal is to perform essential data preparation and analytical queries, covering core concepts such as data aggregation, filtering, subqueries, view creation, and query optimization (indexing).

Deliverables

superstore_schema.sql: Script to create the database and table structure.

superstore_analysis.sql: Collection of analytical queries.
